Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git add: Adds the changed files in our working directory to the staging area of Git.
  • git git commit -m: Commits changes to our repository.
  • git pull origin main: Ensures our branch is up to date.
  • git push origin feature/add-css: Pushes up all of our changes to the branch
